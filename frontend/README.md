 Project Overview

Al-Jazari Industries is a comprehensive B2B corporate website showcasing engineering-driven clean air solutions for the industrial sector across Egypt and the Middle East.

 Key Features

Modern Angular 17+ with standalone components
Responsive Design - Mobile-first approach
Professional UI - Clean, modern interface
SEO Optimized - Meta tags and semantic HTML
Type-Safe - Full TypeScript implementation
Lazy Loading - Optimized performance
Reactive Forms - Contact form with validation



Prerequisites

- Node.js v18.13.0 or higher
- npm v9.0.0 or higher
- Angular CLI




The application will open at `http://localhost:4200/`

 Design System

 Colors
- **Primary Blue**: #1E3A8A (Trust, Engineering)
- **Secondary Teal**: #0D9488 (Innovation)
- **Accent Orange**: #EA580C (Energy, Action)

 Typography
- **Headings**: Poppins (Bold, Modern)
- **Body**: Inter (Clean, Readable)

 Pages

1. **Home** (`/`) - Company overview, products, sectors, partners
2. **About** (`/about`) - Company information and values
3. **Products** (`/products`) - Product categories and details
4. **Sectors** (`/sectors`) - Industries served
5. **Partners** (`/partners`) - Global partnerships
6. **Contact** (`/contact`) - Contact form and information


The build artifacts will be in the `dist/` directory.
